Administración de agrupaciones

Acceso privado con Firebase Authentication. Aquí podrás completar la ficha de cada agrupación, incluyendo sus datos generales, vídeos de YouTube y el libreto.

Inicio Directorio

Acceso administrador

Inicia sesión con tu usuario de Firebase Authentication para acceder a la administración.

Añadir agrupación

Formulario de alta y edición. Los campos de vídeos y libreto se guardan en cada ficha para mostrarse después en la ventana popup del directorio.

Vídeos

Añade tantos vídeos de YouTube como necesites. Cada vídeo permite título, URL y una breve descripción opcional.

Libreto

Introduce las letras o fragmentos de la agrupación. Puedes indicar tipo (Presentación, Pasodoble, Cuplé...), título y texto completo.

Agrupaciones cargadas

Lista centralizada para buscar, editar y borrar registros. El buscador no distingue tildes.
